# Nigeria CO₂ Emissions Analysis

Hypothesis-driven Streamlit app exploring Nigeria’s CO₂ emissions per capita vs global trends, electricity generation mix, and regional context.

- Interactive charts (Plotly + Streamlit)
- Green forest themed background
- Front-end showcase; back-end continues with panel regressions & ML pipelines

## Run locally
python -m venv venv source venv/bin/activate 
# Windows: venv\Scripts\Activate.ps1 pip install -r requirements.txt streamlit run app.py
## Deploy
Push to GitHub and connect on Streamlit Cloud. App auto-redeploys on each commit.
